# WebToSpeech
WebToSpeech is a web-based document reader that lets users upload their documents and have them read aloud via text-to-speech. 
Users will also be able to create collections to organize their documents, keeping related files in one place that can be accessed and viewed within the app. 
WebToSpeech provides users with a more accessible reading experience for those with reading disabilities, and is also a general platform for any user to have their documents read aloud for entertainment, education, or other purposes.

# Project Structure
## webtospeech
This folder contains the WebToSpeech project, built with React. All source code for the project is contained here.

## documents
This folder contains documents related to the project's creation, including our initial requirements draft, posters, and PowerPoints that showcase its purpose.

## Documents/Database_Backup
This folder contains notes on the initialization/restoration of the Supabase database used to run the WebToSpeech project.
